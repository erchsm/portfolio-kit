WAN Automation Engine(WAE)

Cisco WAN Group

WAE is a set of network management API’s for Enterprise level Service Providers—like Comcast, Verizon and AT&T

In my first position at Cisco I worked on creating and documenting REST API’s in the Wide Area Network (WAN) Orchestration product group. The WAN Automation Engine (WAE) is a set of web API’s designed to provide bandwidth orchestration capability for a Wide Area Network in the Service Provider environment.

Utilizing Slate and custom CSS, I created the tutorial seen above, customized to Cisco’s branding.

Our API’s were designed for use by a developer working in an organization that provides network services. Much of my job was figuring out the best way to deliver these API’s to them and make them as easy to use as possible.

In my second position I worked as a UI Engineer on a visual layer to these API’s

Cisco’s Mate, the network visualization web app, allowed users to view their topography, run reports and view statistics.

At Cisco I worked as a UI Engineer. The project I worked on was a visualization tool for the WAE web API’s that I helped to create in my second role there. This role involved utilized HTML, CSS and Javascript–more specifically javascript libraries/frameworks including NodeJS, BackboneJS, D3.js & HighchartsJS.
